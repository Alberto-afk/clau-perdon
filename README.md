# ¿me perdonas? 🥺

Página web minimalista para pedir perdón.

## Cómo agregar el audio

1. Coloca tu archivo MP3 en esta carpeta con el nombre exacto: `perdon.mp3`
2. Sube los cambios a GitHub

## Estructura

```
/
├── index.html
├── style.css
├── script.js
├── perro-triste.png
├── perro-feliz.png
└── perdon.mp3        ← (agregar tú)
```
